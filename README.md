# links_and_files_normalize
Приведение ссылок и файлов сайта к заданным условиям

# Описание
скрипт расположеный в корне сайта и имеющий доступ к файловой системе, сканирует ссылки страниц сайта и если по указанному пути есть файл, то папки и файл переименовываться в соответствии с условиями: 
	1. не более 14 символов в названии; 
	2. символы отличные от букв и цифр должны быть заменены на нижнее подчеркивание (пробелы в том числе); 
	3. если в папке после переименования несколько файлов с одинаковыми названиями, то добавляем уникальный числовой идентификатор. 
	4. Русские буквы должны переименовываться в латиницу.
  5. Ссылки на страницах сайта заменяются на новые переименованные пути.
